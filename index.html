About

A Brief History

  • 2008

    Our Humble Beginnings

    In 2008, Doug Burks started working on Security Onion, a Linux distribution for intrusion detection, network security monitoring, and log management.

  • 2009

    First Release

    A few months later, the first version of Security Onion was released in 2009.

  • 2012

    Big Distro Rebuild (BDR)

    Over the years, folks began deploying Security Onion on production networks with more and more traffic. In 2012, Security Onion was rebuilt from the ground up, focusing on performance and scalability.

  • 2014

    Security Onion Solutions LLC

    As the worldwide Security Onion community grew, Doug Burks began receiving more and more requests for training and professional services. In 2014, Doug started Security Onion Solutions LLC to help Security Onion users peel back the layers of their networks.

  • 2015

    200,000 Downloads

    Security Onion ISO image downloads hit 200,000 in 2015.

  • 2016

    Company Expansion

    Security Onion Solutions LLC expanded in 2016, hiring Phil Plantamura as Chief Operating Officer.

  • 2017

    More Growth

    In 2017, Security Onion Solutions hired Wes Lambert as Senior Engineer. ISO image downloads broke 500,000.

  • 2018

    Growth Continues

    Security Onion Solutions LLC expanded again in 2018, hiring Mike Reeves as Product Manager. We then released our Elastic Stack integration and Security Onion 16.04. We're currently over 600,000 ISO image downloads!
